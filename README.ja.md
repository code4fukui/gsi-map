# gsi-map

国土地理院の地図を表示するWebコンポーネントです。

## デモ
[デモページ](https://code4fukui.github.io/gsi-map/)で確認できます。

## 機能
- 国土地理院の標準地図やサテライト地図の表示
- マップ上の位置座標をGeo3x3形式で表示
- Geo3x3コードをクリックするとGeo3x3のウェブサイトを開く

## 必要環境
特になし。モダンなWebブラウザで動作します。

## 使い方
HTMLファイルにコンポーネントを埋め込むだけで使えます。

```html
<gsi-map center="E913881" mode="satellite"></gsi-map>
```
